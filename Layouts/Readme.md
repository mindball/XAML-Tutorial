# Layouts

## Grid
```
Grid-a заема цялото простанство Пример:
 <Grid>
        <Rectangle Fill="Green"></Rectangle>
 </Grid>
 vs
 <Canvas>
        <Rectangle Canvas.Top="50" Canvas.Left="50" Fill="Green" Height="50" Width="50"></Rectangle>
</Canvas>
```
```
Ако имаме дефиниране редове например 2, и в първия set-нем Height=2*,
първият ред ще заеме 2/3 от целият грид, a вторият ред ще заеме 1/3
```

## Canvas
```
Има абсолютни позиции, това означава че не се знае колко е голямо, 
не заема място по самият screen ние трябва да му определим,
все едно има още един layout върху screen-а и печатаме върху него, 
това означава абсолютни позиции те са независими(не зависят
от други елементи или те са на различни листове) за различка от grid-a
```
